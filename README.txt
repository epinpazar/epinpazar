
EPINPAZAR - Professional Frontend Demo
=====================================

Bu paket:
- Tam responsive frontend (HTML/CSS/JS)
- Ürün listeleme, detay modal, sepet, checkout simülasyonu
- Admin panel (demo) — ürün CRUD, import/export (JSON)
- Otomatik kod teslimi simülasyonu (kodlar ürün.codes listesinden pop edilerek verilir)
- Tüm veriler localStorage'da tutulur (gerçek ödeme ve kullanıcı yönetimi backend gerektirir)

Admin parola (demo): admin123

Kullanım:
1) Zip'i açın ve dosyaları GitHub repo'nuzun root'una atın.
2) GitHub Pages ile yayınlayın veya herhangi bir statik host'a yükleyin.
3) Admin paneli: sağ üstte 'Admin' butonuna basın -> parola ile giriş yapıp ürün ekleyin.

Geliştirme notları:
- Gerçek mağaza için backend (DB, ödeme sağlayıcı entegrasyonu, kullanıcı hesapları) eklenmelidir.
- Otomatik teslimat için güvenli bir sunucu-side sistem gereklidir.
